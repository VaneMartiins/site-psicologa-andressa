Área preparada para um painel editorial.

Por enquanto, os posts são páginas estáticas no código para manter o site leve e estável. Quando for hora de permitir edição sem código, a solução mais adequada para Vercel + GitHub é usar um CMS Git-based.

Solução recomendada

Decap CMS com backend GitHub é uma opção leve para este projeto estático: os textos ficam no repositório GitHub e a Vercel publica automaticamente após cada alteração.

Para funcionar em produção, será necessário configurar autenticação GitHub OAuth por uma função serverless na Vercel, porque o GitHub não deve expor o segredo de autenticação no navegador.
